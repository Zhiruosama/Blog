    Day 1

    第一天入职，先了解了一下项目的大致内容。

    这是一个用 Python + FastAPI 开发的后端项目，整体内容不算很多，主要还是一些 CRUD 相关功能。

    代码放在 GitLab 上。对方给了我一个账号，我就先把代码拉下来读了一遍。

    后续又给了一个共享的 Cursor 账号。后面如果有开发任务，应该主要会用 Cursor 来完成。

    项目架构与技术栈

    • 使用 FastAPI 提供 /api/v1 的 HTTP 接口。
    • 使用 MySQL + SQLAlchemy ORM 存储业务数据。
    • 使用 JWT 做登录鉴权与会话校验。
    • 对接外部 AI Agent 服务（支持 SSE 流式）来完成问答/生成能力。
    • 后端负责请求转发、数据落库和多租户隔离。

    小结

    今天大致就是这些内容。没人带，只能自己先摸索。

    打算后续如果没有安排具体工作，就自学一下 Python，至少不能纯粹浪费时间。
